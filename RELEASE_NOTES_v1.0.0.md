# Sensa v1.0.0

## Highlights

- Native Chrome side panel AI assistant
- Multi-turn chat
- Per-page session persistence
- Optional page context and screenshot per turn
- Streaming response rendering
- Lightweight environment trail
- Compact local persistence to avoid storage quota issues

## Install

1. Download `sensa-v1.0.0.zip`
2. Unzip it locally
3. Open `chrome://extensions`
4. Enable Developer Mode
5. Click `Load unpacked`
6. Select the unzipped folder

## Notes

- Default base URL: `https://api.aixhan.com/v1`
- Default model: `gpt-5.4`
